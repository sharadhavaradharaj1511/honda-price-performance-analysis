# Honda Price & Performance Analysis

## Project Type
Academic Team Project | Baylor University

## Project Overview
Conducted statistical analysis to identify factors influencing Honda car pricing and performance using regression modeling, ANOVA, interaction testing, and business interpretation.

## Tools & Skills Used
- R Programming
- Statistical Modeling
- ANOVA
- Regression Analysis
- Interaction Testing
- LINE Assumptions Testing
- Data Analysis
- Business Insights

## Key Work Performed
- Analyzed factors influencing Honda vehicle pricing
- Performed ANOVA and multivariate modeling
- Conducted interaction testing between variables
- Evaluated LINE assumptions for model validity
- Identified influential points and refitted models
- Generated pricing and performance-related business insights

## My Contribution
Collaborated on statistical modeling, interaction testing, model interpretation, and business insights generation.
